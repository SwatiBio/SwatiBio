## 🚀 Featured Projects

### [Autism SLiM Discovery](https://github.com/swati-bioinfo/Autism-SLiM-Discovery)
Systematic discovery of conserved short linear motifs in intrinsically disordered regions of autism risk genes (SFARI) vs matched controls. MEME discriminative motif discovery, TOMTOM validation, bootstrap analysis, and interactive HTML report.

`Python` `BioPython` `MEME Suite` `IUPred3` `Fisher's exact test`

### [TCGA-BRCA Multi-Omics MOFA2](https://github.com/swati-bioinfo/TCGA-BRCA-Multi-Omics-Project)
Integrating RNA-seq, DNA methylation, and RPPA proteomics from TCGA-BRCA (n=636) using MOFA2. Discovered 15 latent factors linked to breast cancer survival with Cox, RSF, TimeROC, and RMST validation. Interactive Shiny dashboard.

`R` `MOFA2` `UCSC Xena` `Survival Analysis` `Shiny`
